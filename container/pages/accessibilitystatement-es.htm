Invesafe se compromete a hacer que su sitio web sea utilizable por todas las personas al cumplir o superar los requisitos de las Pautas de accesibilidad al contenido web 2.1 Nivel AA (WCAG 2.1 AA). Tenga en cuenta que nuestros esfuerzos continúan a medida que incorporamos las mejoras relevantes para cumplir con las pautas WCAG 2.1 AA a lo largo del tiempo. Si encuentra un problema de accesibilidad, asegúrese de especificar la página web y haremos todos los esfuerzos razonables para que esa página sea accesible.
